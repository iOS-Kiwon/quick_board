# 스컬킹 점수 계산기 — 설계 문서

**작성일**: 2026-04-23  
**배포 대상**: GitHub Pages  
**구현 방식**: 순수 HTML 단일 파일 (`index.html`)

---

## 1. 개요

보드게임 스컬킹(Skull King)의 점수를 계산하는 웹사이트.  
최대 8명이 10라운드 동안 라운드별 점수를 입력하면 총점을 실시간으로 업데이트한다.  
외부 의존성 없이 `index.html` 파일 하나로 동작하며, GitHub Pages에 파일 1개만 올리면 배포 완료.

---

## 2. 화면 흐름

```
[설정 화면]
  └→ [게임 화면]
        └→ [결과 화면]
              └→ [설정 화면] (새 게임)
```

### 2-1. 설정 화면
- 플레이어 수 선택: 2~8명 (셀렉트박스)
- 플레이어 이름 입력: 수에 맞게 텍스트 인풋 동적 생성
- "게임 시작" 버튼 → 게임 화면으로 전환

### 2-2. 게임 화면 (핵심)

**상단 내비게이션**
- 현재 라운드 표시: `← 3 / 10 →`
- 이전/다음 라운드 이동 버튼 (이미 입력된 라운드는 언제든 다시 수정 가능)
- 10라운드 완료 후 "게임 종료" 버튼 활성화

**입력 테이블 (라운드별 입력 폼)**

| 플레이어 | 예측한 승수 | 획득한 승수 | 보너스 점수 | 라운드 점수 |
|---------|-----------|-----------|-----------|-----------|
| 김철수  | [입력]     | [입력]     | [입력]     | 자동 계산  |
| 이영희  | [입력]     | [입력]     | [입력]     | 자동 계산  |
| ...    | ...        | ...        | ...        | ...       |

- 플레이어 = 행, 입력 항목 = 열
- 라운드 점수 열은 입력 즉시 자동 계산되어 표시
- 양수: 초록, 음수: 빨강으로 색상 강조

**누적 총점 표 (항상 화면 하단에 고정)**

| 라운드 | 김철수 | 이영희 | 박지민 | ... |
|-------|-------|-------|-------|-----|
| 1     | +20   | -10   | +40   | ... |
| 2     | +70   | +30   | -20   | ... |
| 합계   | **90** | **20** | **20** | ... |

- 라운드 = 행, 플레이어 = 열 (보드게임 점수표 표준 형식)
- 현재 진행 중인 라운드 행은 하이라이트
- 합계 행은 항상 강조 표시
- 입력값 변경 시 즉시 갱신

### 2-3. 결과 화면
- 최종 순위 표시 (1위 강조)
- 전체 라운드 점수 총람 테이블
- "결과 복사" 버튼: 텍스트 형식으로 클립보드 복사
- "새 게임" 버튼: 설정 화면으로 돌아감

## 3. 점수 계산 로직

라운드 N에서 각 플레이어 입력값:
- **예측한 승수 (Bid)**: 0 이상, N 이하 정수
- **획득한 승수 (Tricks)**: 0 이상, N 이하 정수  
- **보너스 점수 (Bonus)**: 제한 없음 (음수 가능)

```
if bid == 0:
  tricks == 0  →  라운드 점수 = N × 10    (성공)
  tricks  > 0  →  라운드 점수 = N × -10   (실패)
else:
  bid == tricks  →  라운드 점수 = bid × 20              (성공)
  bid != tricks  →  라운드 점수 = |bid - tricks| × -10  (실패)

최종 라운드 점수 = 위 계산값 + bonus
총점 = 1~10라운드 점수 합산 (실시간 갱신)
```

**엣지 케이스**
- 입력값이 비어있으면 0으로 처리 (미입력 라운드는 0점)
- 획득한 승수 최댓값은 라운드 N으로 제한 (HTML `max` 속성)
- 보너스 점수는 성공/실패 무관하게 항상 합산
- 비드/트릭은 음수 입력 불가 (HTML `min=0` 속성)

## 6. 컴포넌트 구조 (JS 내부)

```
state = {
  players: string[],          // 플레이어 이름 배열 (2~8명)
  currentRound: number,       // 현재 보고 있는 라운드 (1~10)
  scores: Score[][],          // scores[playerIdx][roundIdx]
}

Score = {
  bid: number,
  tricks: number,
  bonus: number,
  roundScore: number,         // 자동 계산
}

함수:
  calculateRoundScore(round, bid, tricks, bonus) → number
  renderSetupScreen()
  renderGameScreen()
  renderResultScreen()
  updateScoreboard()          // 누적 총점 테이블 갱신
  exportResults()             // 클립보드 복사
```

## 7. 구현 순서

1. **설정 화면**: 플레이어 수/이름 입력 → 게임 시작
2. **게임 화면 기본 구조**: 라운드 내비게이션 + 입력 테이블
3. **점수 계산 로직**: `calculateRoundScore` 함수 + 실시간 갱신
4. **누적 총점 테이블**: 라운드=행, 플레이어=열 형식
5. **결과 화면**: 최종 순위 + 내보내기
6. **스타일링**: 해적 테마 CSS + 반응형
